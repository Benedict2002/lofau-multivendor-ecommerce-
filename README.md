Lofau Multivendor Ecommerce Platform
Overview
Lofau is a multivendor ecommerce platform that enables multiple sellers to manage products, orders, and customers within a single marketplace.

Tech Stack
Backend: Spring Boot, JPA, Hibernate
Frontend: React, Tailwind CSS, Material UI
Database: MySQL
Authentication: JWT
Version Control: Git & GitHub
Features
Vendor registration & management
Product listing & approval
Order management
Inspection & moderation system
Role-based access control (Admin, Vendor, User)
Project Structure
/backend /frontend

Setup Instructions
Clone repository
Configure application.properties
Run backend
Run fronten
